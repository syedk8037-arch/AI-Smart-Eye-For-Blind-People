AI Smart Eye for Blind People 👓🤖

An AI-powered smart assistive system designed to help visually impaired people navigate and understand their surroundings independently using Computer Vision, Deep Learning, and Raspberry Pi.




📌 Project Overview

AI Smart Eye for Blind People is an intelligent wearable system that uses a Raspberry Pi, camera module, ultrasonic sensors, and AI-based object detection to provide real-time audio feedback to visually impaired users.



The system can:

Detect objects in real time
Recognize faces
Read text aloud
Detect obstacles
Convert text to speech
Assist in navigation

This project aims to improve the independence, safety, and daily life experience of blind and visually impaired individuals.



## 🎯 Objective

The main objective of this project is to help visually impaired individuals navigate safely and independently using AI-powered real-time assistance.

Face recognition identifies known persons
